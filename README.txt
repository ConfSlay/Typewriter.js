
HOW DO THIS WORKS :
	
	1) You need :
		- one or more html elements with class="inputTypewriter"
		- one Typewriter function's output container -> <div class="outputTypewriter"></div>
		- to add an event so the function will start working -> <body onload="typewriter();">


	2) Input elements will be used and then removed to create new elements in the container

	3) thats all ...
